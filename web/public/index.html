Open UK mosque & jamaat timetable data

A public data utility containing UK mosque identities, source provenance, jamaat (congregational prayer) timetables, and freshness status — usable by any Muslim app, community tool, or researcher.

Explore the API Download bulk data

Public read API

Search mosques, fetch published jamaat times by date, find times nearby, and follow an incremental change feed. Every published row carries its source, confidence, and freshness.

Read the API reference →

Bulk exports

Versioned, deterministic snapshots in NDJSON and CSV, plus a change log and attribution manifest. Sync from snapshots instead of calling the API on your request path.

Browse the latest snapshot →

Provenance first

Calculated times are never labelled as jamaat. Restricted partner data never enters public exports. What you see is what a mosque or a trusted source actually published.
